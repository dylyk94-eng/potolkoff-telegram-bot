# Telegram-бот для компании "Потолкоф"

Этот Telegram-бот предназначен для компании "Потолкоф" - студии натяжных потолков, ремонта и дизайна. Бот позволяет пользователям получать информацию об услугах компании, связываться с ними и получать консультации.

## Функциональность

- Главное меню с основными разделами
- Информация об услугах компании
- Контактные данные и способы связи
- Статистика компании
- Подробное описание каждой услуги
- Возможность получения консультации

## Установка

1. Склонируйте репозиторий
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Создайте бота у [@BotFather](https://t.me/BotFather) в Telegram и получите токен
4. Переименуйте файл `.env.example` в `.env` и укажите токен бота:
   ```env
   BOT_TOKEN=ваш_токен_бота_здесь
   ```
5. Запустите бота:
   ```bash
   npm start
   ```

## Использование

После запуска бота вы можете начать с ним взаимодействие через Telegram, найдя его по имени, которое вы указали при создании у BotFather.

## Структура проекта

- `bot.js` - основной файл с логикой бота
- `package.json` - зависимости и настройки проекта
- `.env` - переменные окружения (не включены в репозиторий)

## Технологии

- [Telegraf](https://telegraf.js.org/) - фреймворк для создания Telegram-ботов
- [Node.js](https://nodejs.org/)
- [Express.js](https://expressjs.com/)
