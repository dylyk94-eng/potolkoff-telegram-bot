# Руководство по настройке Telegram-бота для Потолкоф

## Создание бота в Telegram

1. Найдите в Telegram бота [@BotFather](https://t.me/BotFather)
2. Отправьте команду `/newbot`
3. Следуйте инструкциям:
   - Придумайте имя для бота (например, "Потолкоф Бот")
   - Придумайте юзернейм для бота (должен заканчиваться на "bot", например: `potolkoff_helper_bot`)
4. После создания BotFather предоставит вам токен в формате: `1234567890:ABCdefGHIjklMNOpqrsTUVwxyz123456789`

## Настройка токена

1. Откройте файл `.env` в директории проекта
2. Замените `ваш_токен_бота_здесь` на реальный токен, который вы получили от BotFather:

```
BOT_TOKEN=1234567890:ABCdefGHIjklMNOpqrsTUVwxyz123456789
```

## Запуск бота

1. Убедитесь, что вы находитесь в директории проекта
2. Выполните команду:

```bash
npm start
```

## Возможности бота

Ваш Telegram-бот для компании "Потолкоф" предоставляет следующие возможности:

- **Главное меню**: Приветствие и основные опции
- **Контакты**: Все способы связи с компанией (телефон, Telegram, VK, Instagram)
- **Услуги**: Подробная информация о всех предоставляемых услугах:
  - Натяжные потолки
  - Многоуровневые потолки
  - 3D-потолки с фотопечатью
  - Ремонт "под ключ"
  - Дизайн интерьеров
- **О компании**: Информация о студии, преимущества, опыт работы
- **Статистика**: Ключевые метрики компании (объекты, клиенты, опыт, удовлетворенность)
- **Консультация**: Возможность получить персональную консультацию

## Преимущества бота для бизнеса

1. **Круглосуточная доступность**: Бот всегда готов отвечать на вопросы клиентов
2. **Мгновенная реакция**: Быстрое предоставление информации о компания и услугах
3. **Конверсия**: Прямые ссылки для связи, что увеличивает вероятность обращения
4. **Автоматизация**: Снижение нагрузки на менеджеров за счет автоматического предоставления базовой информации
5. **Профессиональный имидж**: Современный способ взаимодействия с клиентами

## Дополнительные возможности

Бот также реагирует на текстовые команды:
- Приветственные слова активируют приветственное сообщение
- Слова "услуги", "работы", "цены" показывают меню услуг
- Слова "контакты", "телефон", "связь" открывают меню контактов

## Поддержка и развитие

Для изменения информации о компании (цены, услуги, контакты) достаточно отредактировать соответствующие разделы в файле `bot.js` в объекте `companyInfo`.

При возникновении вопросов или необходимости дополнительных функций, бот легко расширяем.
